# 헬로우 피카츄

Unity 강의를 들으며 배운 내용을 토대로 간단한 토글 게임을 만들었습니다.

## 언어

- C#

## 기능

- 게임을 시작할 때, 숫자 10 생성
- 터치를 10번 하면 피카츄가 나옴

## 게임 플레이 방법

- 포켓볼을 터치
